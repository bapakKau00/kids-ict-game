# ICT Adventure for Kids! 🚀🛡️🎮

Selamat Datang! **ICT Adventure** is a fun and interactive web-based game designed to teach children about basic computer hardware, safety, and troubleshooting in a playful and engaging way.

## 🌟 Features
- **Kenali Jenis Komputer**: Use a virtual flashlight to discover hardware components and hear their names!
- **Protect Komputer**: A fast-paced minigame where you catch valid computer parts and avoid distractions.
- **Repair Komputer**: Learn how to wire up a desktop PC and assemble a digital camera.
- **Interactive Audio**: Authentic voice-overs, background music, and celebratory sound effects.
- **Seamless Navigation**: Smooth "iris-zoom" transitions between all game levels.

## 📁 Project Structure
- **/assets/images/**: All colorful background images and character icons.
- **/assets/audio/**: Voice-overs, sound effects, and background music.
- **/js/**: Game logic and audio controllers.
- **/css/**: Global styles and animations.
- **Root**: All HTML game stages (`index.html`, `kenali.html`, etc.).

## 🚀 How to Run
Simply open `index.html` in any modern web browser to start the adventure!
